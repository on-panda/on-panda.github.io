onPanda

Efficient Annotation of On-Policy Alignment Data for LLMs and Agents via Token-Level Correction


Authors: comming soon

Paper: comming soon | Code | Demo: comming soon | Dataset: comming soon


Abstract

We present onPanda, an interactive tool for efficiently annotating LLM alignment data and agent trajectories. While reading a model response, the annotator locates the first inappropriate token and either selects a substitute from the model's candidates or enters a free-form correction. The system then truncates the subsequent content and continues generation from the corrected prefix, repeating this locate-correct-continue loop until a satisfactory response is obtained. Most tokens therefore remain generated by the rollout model, while each correction automatically records precise positions and naturally paired positive-negative samples for fine-grained supervision.


Token-Level Correction


The token-level correction interface of onPanda. The annotator locates an inappropriate token, selects a better candidate or enters a free-form edit, and lets the model continue from the corrected prefix. Intermediate versions are retained automatically.


Agent Trajectory Annotation


Annotating an agent trajectory with onPanda. Reasoning and tool-call arguments remain editable at token level. Corrected tool calls can be executed in the connected environment, and the resulting trajectory continues from the corrected context.
